# Weekly Review Playbook

Friday 17:00 IST. ~45 minutes operator time. Closes the week, plans the next.

## Trigger

- Calendar event "Weekly Review" Friday 17:00 IST
- Or "run weekly review"

## Sequence

### 1. P&L pass (marketing-analytics)
- Revenue, gross margin, ad spend, RTO loss, contribution margin — week + last 8 weeks
- By product (top 10) and by channel
- New vs repeat customer mix

### 2. Product performance (marketing-analytics + product-research)
- Top 5 winners — should we scale further? hand to `scale-winner` playbook
- Bottom 5 — kill or refresh? hand to `kill-loser` playbook
- 3 candidates from research pipeline ready to launch next week

### 3. Creative review (creative-studio)
- Top 5 ads by ROAS — extract patterns (hook, format, talent, music)
- Bottom 5 — what failed pattern
- Refresh queue size for next week

### 4. Operations review
- RTO % vs target — `customer-support` + `order-fulfillment` collaborate
- Average response time on customer emails
- Any supplier or courier issues

### 5. Cash flow check (operator + ops-planner)
- COD remittance vs spend ratio (target ≥1.4 weekly)
- 30-day operating buffer status
- Tax/GST provision

### 6. Plan next week (ops-planner)
- Launch calendar: which products, which days, which creatives due when
- Festival prep status for upcoming windows (see `indian-dropshipping` festival calendar)
- Notion "Launches" DB updated, Calendar events created

### 7. Wins / Problems / Experiments (Notion log)
- 3 wins (celebrate the operator)
- 3 problems (must-fix this week)
- 3 experiments (new hooks, new audiences, new SKUs)

## Operator deliverable

- One-page summary in Notion "Weekly Reports" with last 8 weeks of data
- Next week's launch and creative calendar finalized
- 3-3-3 decided

## Hard rule

If gross margin trend is down 3 consecutive weeks, halt all scale moves and run a margin diagnostic before next launch.
